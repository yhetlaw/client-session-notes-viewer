# Client Session Notes Viewer

Client therapy session notes with real-time updates and responsive UI.

## Built With

- **TypeScript**
- **React Router v6**
- **Axios**
- **React Query**
- **React Hook Form**
- **Material UI**

## Run Locally

- Clone repository
 
- Frontend:  
1. `npm i`
2. `npm run dev`

- Backend:
1. `json-server --watch db.json --port 3001`
